# Free AI Agent Starter Kit

Welcome — you've downloaded the starter kit for the Zaheer Studio system.

This is the bare-bones template that teaches you how Claude Code agents work in the Zaheer system. Use it as a foundation for your own agents, or as a learning sandbox.

## What's in this pack

- `agent.md` — the core agent definition
- `skills/` — one example skill to copy and customize
- `SETUP.md` — 10-minute setup guide

## Quick start

1. Drop this folder anywhere in your projects directory
2. Open it in Claude Code
3. The agent definition loads automatically
