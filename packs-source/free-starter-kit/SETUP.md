# Setup — 10 minutes

## 1. Drop the folder

Move this `free-starter-kit/` folder anywhere in your projects directory. A common spot is `~/Documents/`.

## 2. Open in Claude Code

```
cd ~/Documents/free-starter-kit
claude
```

The agent definition in `agent.md` loads automatically.

## 3. Try it

Ask the agent: "What can you help me with?"

That's it. You're running.

## Next: build your own skill

Open `skills/example-skill.md` and use it as a template. Copy it, rename it, and edit the prompt to do whatever you need.
